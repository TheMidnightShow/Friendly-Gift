# Friendly-Gift
Site desenvolvido com participação na criação do HTML e CSS, com o objetivo de servir como uma ponte direta entre empresas solidárias e doadores.

O projeto foi pensado para facilitar a doação de alimentos, produtos de higiene, roupas e outros itens essenciais, especialmente em momentos de emergência.

Ele foi criado durante o período das enchentes, como parte do programa EnduEmpren, no qual jovens desenvolviam produtos e soluções com impacto social.
